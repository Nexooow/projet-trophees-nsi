# projet-trophees-nsi

Projet réalisé dans le cadre du concours "Les trophées NSI"

## Installation 

```bash
git clone https://github.com/Nexooow/projet-trophees-nsi.git
cd projet-trophees-nsi
pip install -r "requirements.txt"
```

## Démarrer le jeu

```bash
python3 src/main.py
```
